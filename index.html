Twidder

Description

This project features a reconstruction of the populair social media platform Twitter. The project is currently under construction.

In the future users will be able to register and sign up. Once logged in users will have the ability create posts which can be read and commented upon by followed users. The following of other users gives access to their created posts.

A timeline containing updates of the latest posts by users that are followed will be the centering stage of the project.
